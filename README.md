## âš¡ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
- A Firebase project set up.

### Installation

1.  **Clone the repo**
    ```sh
    git clone https://github.com/your_username/tracker_app.git
    ```
2.  **Install dependencies**
    ```sh
    cd tracker_app
    flutter pub get
    ```
3.  **Firebase Setup**
    - Install the [Firebase CLI](https://firebase.google.com/docs/cli).
    - Run `flutterfire configure` to generate `firebase_options.dart` for your project.
    ```sh
    flutterfire configure
    ```
4.  **Run the app**
    ```sh
    flutter run
    ```
